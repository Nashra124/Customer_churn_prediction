# Customer Churn Prediction

## Project Overview

This project predicts customer churn using machine learning models.

## Dataset

The dataset contains customer demographics, account information, billing details, and churn status.

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn
- XGBoost

## Workflow

1. Data Cleaning
2. Data Preprocessing
3. Feature Engineering
4. Handling Imbalanced Data using SMOTE
5. Model Training
6. Model Evaluation

## Models Used

- Logistic Regression
- KNN
- Decision Tree
- Random Forest
- AdaBoost
- Gradient Boosting
- XGBoost
- SVM

## Results

Best Model: AdaBoost

Accuracy: 80.07%
